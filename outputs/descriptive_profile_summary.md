# Descriptive profile summary

Date accessed: 2026-05-14

This section describes the observed national country-year data from Eurostat table `hlth_silc_08`.

The primary outcome file contains 608 observed country-year cells for 37 national country codes from 2008 to 2025. EU and euro-area aggregate codes are not included.

The year with the highest observed cross-country mean is 2013, with a mean of 4.33 percent. This comparison uses the countries observed in each year, so it should be read together with the missingness tables.

The five countries with the highest mean observed values are: AL, EE, LV, EL, XK. The five countries with the lowest mean observed values are: NL, AT, MT, LU, CZ.

The five highest latest available country values are: EL, AL, FI, EE, LV. The five lowest latest available country values are: CY, MT, CZ, NL, TR.

The robustness comparison uses 154 common country-year cells where both `hlth_silc_08` and `hlth_silc_08b` are observed from 2021 to 2025. These indicators have different denominator definitions, so the table is a descriptive comparison only.
