# Country and year indicator summary

The indicator extension uses 282 complete country-year rows from 30 countries.
The model includes country and year indicators. The R-squared is 0.895.

Control coefficient patterns compared with the pooled model:
- `log_gdp_per_capita` changes from -0.546 in the pooled model to -1.041 in the indicator model.
- `unemployment_rate_pc` changes from 0.160 in the pooled model to -0.034 in the indicator model.
- `poverty_or_social_exclusion_pc` changes from 0.062 in the pooled model to 0.177 in the indicator model.
- `government_health_expenditure_gdp_pc` changes from 0.159 in the pooled model to -0.229 in the indicator model.
- `compulsory_health_financing_gdp_pc` changes from -0.401 in the pooled model to 0.097 in the indicator model.

The estimates remain descriptive associations in aggregate country-year data.
They should not be read as evidence about cause and result.
